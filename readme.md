# Jäätelön klikkaus

Klikkaa jäätelöä ja hanki lisää jäätelöitä.

## Miten pelata
- Klikkaa jäätelöä
- Osta kaupasta apuvälineitä
- Saa jäätelöitä automaattisesti

## Tekniikka
- HTML, CSS, JavaScript
- Tallennus selaimeen

## Tehtävät
- [x] Klikkaus toimii
- [x] Kauppa toimii
- [x] Automaattinen tallennus
- [ ] Lisää ostoksia
- [ ] Äänet

## Käyttö
Avaa `index.html` selaimessa.
